Display available coupons in product page.

== Description ==

This plugin adds a feature to display available coupons on the product page of a WooCommerce site. When a user navigates to a product page, this plugin scans for any available coupons and displays them in a list on the product page.

This allows the user to easily see any discounts or promotions that are currently available for that product, and they can then use the coupon code at checkout to receive the discount.

This plugin inspired by coupon listing from [Zalora](https://www.zalora.com.my/) e-commerce site.

== Installation ==

1. Log in to your WordPress admin.
2. Search for **Coupon Listing for WooCommerce**.
3. Install and activate the plugin.
5. Navigate to "Plugins" in the sidebar to access the plugin list page.
6. Find "Coupon Listing for WooCommerce", then click "Settings" to access the plugin settings page.
7. Follow the instructions and update the plugin settings.
